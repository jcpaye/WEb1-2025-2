Propiedad Display

La propiedad display especifica el tipo de cuadro de representación utilizado para un elemento HTML. Es una de las propiedades más importantes para controlar el diseño.

Valores Principales

Display: Block

Los elementos con display: block ocupan todo el ancho disponible y comienzan en una nueva línea.

Propiedad Box-Sizing

La propiedad box-sizing define cómo se calculan el ancho y alto total de un elemento, incluyendo o excluyendo relleno (padding) y bordes (border).

Box-Sizing: Content-Box (Valor por defecto)

El ancho y alto especificados solo se aplican al contenido del elemento. El padding y border se añaden por fuera.

Content-Box

Ancho definido: 100%

Padding: 15px

Border: 5px

Ancho total: 100% + 30px (padding) + 10px (border)

Border-Box

Ancho definido: 100%

Padding: 15px

Border: 5px

Ancho total: 100% (incluye padding y border)

Box-Sizing: Border-Box

El ancho y alto especificados incluyen el contenido, padding y border. Es más intuitivo para calcular dimensiones.

/* Aplicar border-box a todos los elementos */
* {
  box-sizing: border-box;
}

/* O aplicarlo a elementos específicos */
.mi-elemento {
  box-sizing: border-box;
  width: 300px;
  padding: 20px;
  border: 5px solid #ccc;
  /* El ancho total será de 300px */
}
